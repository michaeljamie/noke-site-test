NO SMART DEVICE?

NO PROBLEM

NOKĒ FOB

Have employees without smart devices? Don't want your crew to rely on their phones for access? The Nokē Fob is a Bluetooth enabled, trackable key that allows users to unlock Nokē locks without the use of a smart phone or Quick-Click code.

ACCESS MANAGERS HAVE COMPLETE CONTROL

Using the Nokē software and mobile app, access managers control the fob with the same features a smart device provides

Fob set up

Grant or deny lock access

Create users and groups

Customize schedules

Manage account and security settings

View lock history, activity, and receive alerts

View lock location and track lock activity

Monitor and manage audit trails

Monitor lock battery life

USING A FOB

IS SECURE, FAST, AND EASY

See why the Nokē Fob helps streamline access control when smart devices are not available

SIMPLE TO OPERATE

Press down on the shackle, and squeeze fob while holding it near the lock

UNLIMITED FOBS

A manager can issue as many fobs needed to control a single lock

CAN BE PRIMARY UNLOCKING METHOD

When a manager provides a user with an authorized fob, it can be used as a sole method of unlocking Nokē locks

CONTROL UP TO 100 LOCKS

100 locks can be controlled by a single fob
